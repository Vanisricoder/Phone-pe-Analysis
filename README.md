# Phone-pe-Analysis
# 📱 PhonePe Transaction Analytics

An end-to-end analytics project aimed at understanding transaction patterns, service usage, customer demographics, and growth trends using Python, MySQL, and Power BI.

## 🚀 Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- MySQL
- Power BI
- DAX

## 📊 Key Features
✔ Data Cleaning & EDA  
✔ Multi-sheet Data Integration  
✔ MySQL Database Storage  
✔ DAX Measures & KPI Analysis  
✔ Interactive Power BI Dashboard

## 📈 Dashboard Highlights
- ₹426.16M Transaction Value
- 199K Transactions
- 96.08% Success Rate
- MoM Growth Analysis
- Service-wise Transaction Trends
- User Demographics Analysis

Python • MySQL • Power BI • DAX • EDA • Data Visualization • Business Intelligence
